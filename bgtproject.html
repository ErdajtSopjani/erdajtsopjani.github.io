Details about this project:

Me and two of my classmates got invited to a tech competition called "Java e Shkences".

We decided to do a big project that will change the world one day.

This project can automatically detect any CARS, HANDS, TRAFFIC SINGS AND HUMAN BODIES that are in frame of the camera.

Im confident this project and these technologies will have a really big impact on our daily lives.

Everything here was made with Python even the GUI, exept the website of the project.

You can try out this project yourself as well like every other project listed on my CV.

We won first place with this project.
